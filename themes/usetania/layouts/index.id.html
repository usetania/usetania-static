Tania berbasis komunitas-kolektif

Tania adalah perangkat lunak untuk manajemen pertanian bersumber terbuka yang dihost di GitHub, diprakarsai oleh Tanibox sejak tahun 2017. Komunitas Tania terdiri dari berbagai orang yang berdedikasi tinggi seperti pengembang perangkat lunak, pengguna, petani dan ahli agrikultur yang membuat sistem ini makin baik. Mereka berkontribusi dan membagikan pengalaman serta pengetahuannya melalui:

  • GitHub

  • Telegram

Dibangun untuk IoT

Ingin menghubungkan sensor iklim dan pengatur irigasi supaya tetap bisa dikendalikan dari jauh saat sedang pergi keluar kota? Tenang saja, Tania dibangun untuk pecinta IoT seperti kamu!

Dapat Diperpanjang

Modul plugin yang dibangun untuk menyesuaikan sistem Tania dengan kebutuhan. Anda juga dapat membuat sensor untuk kebutuhan bisnis dan pengawasan lahan Anda sendiri.

open source

Tania akan selamanya gratis dan berupa kode sumber terbuka untuk memberi Anda kebebasan mengelola lahan sesuai keinginan. Kodenya tersedia di GitHub jika Anda ingin berkontribusi.
