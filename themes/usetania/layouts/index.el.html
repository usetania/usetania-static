Το Tania καθοδηγειται απο τη συλλογικη κοινοτητα

Το Tania είναι ένα λογισμικό διαχείρισης αγροκτήματος ανοιχτού κώδικα το οποίο φιλοξενείται στο GitHub, και ξεκίνησε απο το Tanibox το 2017. Η κοινότητα Tania αποτελείται απο αφοσιωμένους προγραμματιστές, χρήστες, αγρότες, ερευνητές και ειδικούς στη γεωργία οι οποίοι το έκαναν πραγματικότητα. Για να συνεισφέρετε, να μοιραστείτε τη γνώση και τις εμπειρίες, μπορείτε επισης να μας ακολουθήσετε στα:

  • GitHub

  • Telegram

Συμμετέχοντες

Το Tania είναι περήφανα μέλος της οικογένειας Open Collective.
Μπορείτε να στηρίξετε την ανάπτυξη με το να γίνεται υποστηρικτής ή σπόνσοράς μας.

Οικονομικοί Υποστηρικτές

Γιατί το Tania είναι καλό για Προγραμματιστές και Αγρότες;

κατασκευασμενο για iot

Σκέφτεστε να συνδέσετε αισθητήτες κλίματος και ελεγκτές άρδευσης τους οποίους να μπορείτε να διαχειρίζεστε απομακρυσμένα όταν βρίσκεστε μακριά; Σας έχουμε καλυμένους. Το Tania δημιουργήθηκε για ανθρώπους ενθουσιαμένους με το IoT όπως εσείς!

επεκτασιμο

Επεκτάσιμα δομοστοιχεία για να βοηθήσουν το δικό σας σύστημα Tania να ικανοποιήσει τις ανάγκες σας. Μπορείτε να δημιουργήσετε τις δικές σας τυποποιημένες διαδικασίες λειτουργίας που ταιριάζουν στη δική σας αγροτική επιχείρηση.

ανοιχτος κωδικας

Το Tania θα είναι πάντα δωρεάν και ανοιχτού κώδικα για να σας παρέχει την ελευθερία να διαχειριστείτε το αγρόκτημά σας όπως εσείς επιθυμείτε. Είναι διαθέσιμο στο GitHub.
